Can I add several series at once?
Yes. Separete each series with an empty line and copy to Input area. Click "Add Dummy Data" for example
I don't like "series_1", "series_2" name. Can I change them?
Yes. Click on the series name in the "Data view", change the name and hit "Enter". You will see the new name on new figures
Why even bother with yet another online plotting tool?
Other tools ask to input values one by one or load csv file. It is too long for such a quick task if you have values from some script or program.
